type "npm install" to download all dependencies.
type "npm install -g gulp" to install gulp globaly.
type "npm install gulp" to install gulp for project.

type "gulp deploy" for client application initialization

if "gulp deploy" was executed once, you may type one of
"gulp js" for js transformations only
"gulp css" for sass to css transformation only
"gulp copy" for index.html file copy
"gulp" for js and css tasks (without copy)
